== Description ==

Tweetly Updater is a Wordpress plugin that updates Twitter when you create or edit a blog entry, it uses the bit.ly service for short urls.
It can use your tags and categories to add shashtags to your tweets. You can optionally include pages in your updates.

== Installation ==

Tweetly Updater requires PHP 5 >= 5.2.0 and the curl module for PHP.

* Unzip the release archive to your Wordpress plugin directory
* Activate the plugin on the plugin management page
* Go to _Settings->Tweetly Updater_ and fill in your twitter an bit.ly login information. 
* Enjoy...
